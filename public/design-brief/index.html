Personality (3 rails)

Confident

We replace phone tag with a real booking. No hedging copy.

Calmly clinical

Adjacent to healthcare without being cold. Generous whitespace, low-saturation color, soft shadows.

Respectfully Egyptian

Arabic-first. No Pyramids/Sphinx clichés. No "ancient" anything.

What to avoid (visual references)

Vezeeta look — too "medical blue" undraw style — too generic blob.style gradients — overused
Linear — restraint Stripe — clarity Practo — regional clinical Vercel — depth

D · Iconography

Custom icons + map pin + trust badges.

Lucide handles generic UI icons (chevrons, copy, share). Anything dental-specific is custom. Match Lucide's grid (24×24) and stroke (1.5px) so they slot in next to it.

D.0System rules

24×24 viewBox · 1.5px stroke · round caps/joins · single currentColor (parent sets it). Outline-only. Provide one icons.tsx with React-component exports.

Review process

Round 1 (concept): wireframes / sketches. Confirm direction, ≤3 days turnaround.
Round 2 (production): final files, named per above, in folder structure → engineering integrates.
Sign-off: engineering lead (slots into code unmodified) + founder (on-brand). Founder breaks ties.
